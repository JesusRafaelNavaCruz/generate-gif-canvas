🕒 API de Cuenta Regresiva 🎉

¡Bienvenido a la API de cuenta regresiva! Esta API te permite generar un GIF animado que muestra una cuenta regresiva personalizada.

ℹ️ Uso de la API:

Para generar tu cuenta regresiva personalizada, simplemente realiza una solicitud GET al siguiente endpoint: 

https://generate-gif-canvas-givu84mu4-jesus-navas-projects.vercel.app/?name=TU_NOMBRE&expirationDate=FECHA_EXPIRACION

Reemplaza TU_NOMBRE con tu nombre o el texto que quieras que aparezca en la cuenta regresiva, y FECHA_EXPIRACION con la fecha de vencimiento en formato YYYY-MM-DD HH:MM:SS.

🎨 Ejemplo:

Por ejemplo, si deseas generar una cuenta regresiva para un evento llamado "¡Gran Venta!" que finaliza el 31 de diciembre de 2024 a las 23:59:59, realizarías una solicitud GET a:

https://generate-gif-canvas-givu84mu4-jesus-navas-projects.vercel.app/?name=Gran%20Venta!&expirationDate=2024-12-31%2023:59:59
